Règles à Respecter

Si vous voulez passer une bonne soirée, il est indispensable de respecter ses règles suivantes:

Le respect de la personne, zéro baston

0 jugement t'es là pour passer une bonne soirée avec tout le monde sinon tu restes chez toi devant tiktok.

Respect de l'environnement, zéro pollution

Pas de déchets par terre, les mégots les plastiques dans les poubelles qu'on installera. Vos déchets → poubelle

Objets perdus? Envoyer un message sur insta

Imagine c'est toi tu perds tes affaires tu seras bien contend de les retrouver. Tu veux d'autres affaires? Va bosser.

Transport et achat d'alcool

Si vous avez besoin d'aide pour les transports ou l'achat d'alcool, la solidarité simpose mais après fait pas ton rat, si tu veux boire tu payes poto.
